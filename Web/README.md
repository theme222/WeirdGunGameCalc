# wggcalc-web

This is the web implementation for WeirdGunGameCalc. You can find it here: [https://wggcalc.tongsima.com](https://wggcalc.tongsima.com)

## About

This project uses WebAssembly to run c++ code in the browser. It was built on
the Vue.js framework and it is a frontend only application and all computations
run locally on your browser. 

(The command I used to compile the c++ can be found in the Makefile at the root of this repo.)
